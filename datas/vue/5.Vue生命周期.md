# Vue 生命周期

每个 Vue 实例在被创建之前都要经过一系列的初始化过程,这个过程就是 vue 的生命周期。

请看官网上的图：

![vue生命周期-VgOuyU](https://cdn.jsdelivr.net/gh/DreamCats/imgs@main/uPic/vue生命周期-VgOuyU.png)

可以看到在 vue 一整个的生命周期中会有很多钩子函数提供给我们在 vue 生命周期不同的时刻进行操作, 那么先列出所有的钩子函数:

- beforeCreate
- created
- beforeMount
- mounted
- beforeUpdate
- updated
- beforeDestroy
- destroyed
